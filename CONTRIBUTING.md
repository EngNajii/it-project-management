# 🤝 Contributing

Thanks for your interest in improving this repository! Contributions are welcome and appreciated.

## How to Contribute

### 1. Suggest a New Template or Resource
Open an Issue with the label `enhancement` and describe what you'd like to see added.

### 2. Improve Existing Content
- Fork the repository
- Make your changes
- Submit a Pull Request with a clear description of what you improved and why

### 3. Report an Error
Found a mistake or outdated content? Open an Issue with the label `bug`.

## Guidelines

- Keep templates practical and format-agnostic (Markdown preferred)
- Avoid tool-specific templates (e.g., Jira-only or Asana-only) unless clearly labeled
- Test your templates on a real or realistic project before submitting

## Code of Conduct

Be respectful, constructive, and helpful. This is a professional community.
